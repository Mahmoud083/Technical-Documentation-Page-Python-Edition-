Introduction

Python is a beginner-friendly programming language that's powerful, readable, and versatile.

It supports multiple paradigms: object-oriented, functional, and procedural programming.

Used in web, data science, AI, automation, game dev, and more.

It has a huge standard library and thousands of third-party packages.

First released in 1991 by Guido van Rossum.

print("Hello, Python World!")
Variables and Data Types

Variables store data ‚Äî no type declaration required!

  • int ‚Äî whole numbers
  • float ‚Äî decimal numbers
  • str ‚Äî text values (strings)
  • bool ‚Äî True or False values
  • list, tuple, set, dict ‚Äî built-in collections

Use `type()` to check a variable's type.

Python is dynamically typed but supports type hints.

Everything in Python is an object.

x = 10 pi = 3.14 name = "Leon" is_cool = True
Control Flow

Python uses indentation for blocks ‚Äî no braces!

Supports if/elif/else and logical operators (and/or/not).

For loops iterate over any iterable (list, dict, string).

While loops execute based on a condition.

Use break/continue to control loops.

if score > 90: print("Excellent!") elif score > 70: print("Good!") else: print("Keep going!")
Functions

Functions use `def` to declare reusable blocks of logic.

  • Use `def` to define a function
  • Can accept parameters
  • May return a value
  • Can use default arguments
  • Support *args and **kwargs

Lambda functions are used for short, anonymous functions.

Functions are first-class objects (can be passed around).

def greet(name): return f"Hello, {name}!"
Modules and Imports

Python files can be imported as modules.

  • Use `import` to include modules
  • Standard examples: `math`, `random`, `os`, `sys`
  • Use `from module import x` for specific items
  • Third-party libraries are installed via `pip`
  • Modules can be custom-made or pre-installed
import math from math import sqrt import numpy as np
